MathML

Mathematical Markup Language er et språk for å uttrykke mattematikk på internett. Det skrives på samme måte som HTML kode, med tagger og attributter. Før denne teknologien, brukte man bilder for å illustrere matematiske formler, nå så kan man skrive de i html. Trykk for å lese mer om MathML og se eksempler.s

SVG

Scalable Vector Graphics som er et XML-basert format, brukes til å lage illustrasjoner i HTML. Det kan brukes til å for eksempel lage sirkler, rektangler osv. Det finnes flere fordeler å bruke SVG enn å bruke vanlige bildefiler. Trykk for å lese mer om SVG, og for å se eksempler.

Git og GitHub

Git er et verktøy man bruker for versjonskontroll og deling av kode. Det gjør det enklere for flere personer å jobbe på det samme prosjektet. GitHub er et program som bruker et slikt verktøy. Trykk for å lese mer om Git og GitHub.
